Color Game


Principe du jeu

Le but du jeu pour le joueur 1 est d'aligner 4 case de différentes couleur ( R / V / J / B ) peut importe l'ordre.

Le but du jeu pour le joueur 2 est de tout faire pour empécher le joueur 1 de gagner mais, il n'a pas le droit de jouer à côté du dernier placment du joueur 1.

Le jeu s'arrete quand: soit le joueur 1 gagne (4 couleur différentes sont aligner) soit le joueur 2 gagne (le joueur 1 n'as pas pu aligner 4 cases de couleur différentes).

Comment utiliser le jeu?: cliquez sur l'un des 4 boutons ( V / R / J / B ) qui ce situent en dessous du tableau de jeu puis cliquez sur l'une des cases du plateau de jeu où vous voulez placer la couleur
